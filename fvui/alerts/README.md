# Alerts

## Alert - Danger
**name:** Alert - Danger

**description:** Alert message to indicate danger
``` html
<--! start: bs-4.4.1 alert-danger -->
  <div class="alert alert-danger" role="alert"> This is a danger alert—check it out! </div>
<-- end: bs-4.4.1 alert-danger -->
```
